# Extract parameters from a phase sub-vector (internal scale)

Extract parameters from a phase sub-vector (internal scale)

## Usage

``` r
.hzr_unpack_phase_theta(theta_j, phase)
```

## Arguments

- theta_j:

  Numeric sub-vector for one phase.

- phase:

  An `hzr_phase` object.

## Value

Named list with `log_mu`, `log_t_half`, `nu`, `m`, `beta`.
