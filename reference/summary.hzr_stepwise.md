# Summary method for `hzr_stepwise`

Produces the standard
[`summary.hazard()`](https://ehrlinger.github.io/temporal_hazard/reference/summary.hazard.md)
on the final fit, with the selection trace prepended in the print
method.

## Usage

``` r
# S3 method for class 'hzr_stepwise'
summary(object, ...)
```

## Arguments

- object:

  An `hzr_stepwise` object.

- ...:

  Unused.
